# Android Mimarisini Anlamak
Android’in genel mimarisi beş ana katmandan oluşan bileşenleri içerir:
- Android applications 
- Android Framework 
- Dalvik virtual machine 
- user-space
- Linux kernel

![alt text](resim.png)

Android uygulamaları, geliştiricilerin cihazın alt düzeylerini değiştirmeden cihazın işlevselliğini genişletmesine ve geliştirmesine olanak tanır.
Android Framework ise, geliştiricilere bir Android cihazın sunduğu tüm olanaklara erişim sağlayan bir API sunar. Yani uygulamalar ile Dalvik arasında iletişimi sağlar. 

Hem android yazılımları hem android framework java ile geliştirilir ve DalvikVM üzerinde çalışır.
